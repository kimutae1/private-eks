## 콘솔에서 OIDC 추가 진행 
```
 AWS Management Console을 사용하여 OIDC 제공자를 생성하는 방법은 다음과 같습니다:
 1. IAM 콘솔을 엽니다.
 2. 왼쪽 탐색 창에서 "자격 증명 공급자"를 선택합니다.
 3. "공급자 추가"를 선택합니다.
 4. "OpenID Connect"를 선택합니다.
 6. 대상(Audience)으로 "sts.amazonaws.com"을 입력합니다.
 7. "공급자 추가"를 선택합니다.
 이 방법을 사용하면 VPC 내부에서도 OIDC 제공자를 생성할 수 있습니다.
```
